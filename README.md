# Big Tech Data Misuse and Cyber Security Awareness

## Overview

This repository is an educational cyber security and digital privacy project focused on:

- Big tech surveillance
- User tracking systems
- Data privacy risks
- Behavioral profiling
- Artificial Intelligence analytics
- Metadata collection
- Data breaches
- Social engineering attacks
- Privacy protection methods

The goal of this repository is to improve awareness about how modern digital platforms collect, analyze, store, and process user information.
